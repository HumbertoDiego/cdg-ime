REQUISITOS:

- [uv](https://docs.astral.sh/uv/getting-started/installation/)
- `uv python install 3.14`
- `uv sync`
- Para ativar no terminal:

  - `./.venv/Scripts/activate.ps1 # troque .ps1 dependendo do SO ou terminal`
- Feche o terminal e, se necessário, reinicie o VSCode
- Para ativar neste notebook:

  - `VSCode` > `Select Kernel` > `Python Environments...` > `cdg-ime (3.14.4)(Python 3.14.4) .venv\Scripts\python.exe`
